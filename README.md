# 🏠 House Rent Prediction using Machine Learning

## 📌 Project Overview
This project predicts house rent prices using Machine Learning techniques.

## 🔧 Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 🤖 Models Used
- Linear Regression
- Polynomial Regression
- Ridge Regression

## 📊 Key Steps
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Model Training & Evaluation
- Cross Validation

## 📈 Results
- Polynomial Regression improved accuracy
- Initial overfitting issue detected
- Fixed using proper train-test split and validation

## 🎯 Conclusion
The final model generalizes well and provides reliable predictions.
